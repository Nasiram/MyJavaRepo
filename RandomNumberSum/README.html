#

Random User Validation project

Step1: Build the project

    Execute batch file App-Build

Step2: Run the project

    Execute batch file App-Start

Step3: Check the APIs using Postman. Open Postman application and follow below scenarios

Note: Attached RandomTest.postman_collection

Request Question

Http Method : GET
URL : localhost:8080/requestQuestion

Answer Question

Http Method : GET
URL : localhost:8080/answerToQuestion?answer=Give your value
If response is 200 then user is not a robot as it have validation for previously given values, in which gives an calculated value. Otherwise it gives 400 status code, which is invalid entries!
